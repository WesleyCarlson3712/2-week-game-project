# Overview

I made this game as a 2 week project to see how far i could get using python arcade in a limited time. I have spent many hours making this and i have learned a lot in the process.
This game is a battle simulator, where you control multiple characters that fight against enemy characters. The game takes place on a hexagon board, and the user input is taken primarily from a menu on the right of the screen. The controls are pretty self explanitory, all you need is your mouse. Hovering over different things will display information about them. The game is not fully complete or polished, but this is what i have finished in my 2 week deadline.
